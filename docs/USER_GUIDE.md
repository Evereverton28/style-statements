# User Guide

## Customers
Browse the storefront at `/`. Filter and sort in **Shop**, open a product for
details and reviews, add to cart or wishlist, and check out. Choose M-Pesa, card,
or cash on delivery at checkout. Chat/order via the floating WhatsApp button.

## Admins
Go to `/admin` and sign in with an admin account. From the dashboard you can:
- watch live revenue, orders, customers, and low-stock alerts;
- add / edit / delete products;
- move orders through the pipeline (Pending → Delivered);
- approve, reject, or delete reviews;
- preview permissions with the "View as" role switcher.
